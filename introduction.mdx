# Welcome to EduLearn Hub

EduLearn Hub is a full-stack educational platform built as a **pnpm monorepo** using TypeScript throughout. It provides an Express 5 API server, a PostgreSQL database layer with Drizzle ORM, generated client libraries, and interactive React-based learning experiences.

## What's included

EduLearn Hub is organized as a monorepo with clearly separated concerns across deployable artifacts and shared libraries.

### Artifacts

| Package | Description |
|---------|-------------|
| `@workspace/api-server` | Express 5 REST API server with OpenAPI 3.1 specification |
| `@workspace/mockup-sandbox` | Vite-powered React component preview server for UI development |
| `@workspace/pi-day` | Interactive Pi Day educational app with visualizations and memory games |

### Shared Libraries

| Package | Description |
|---------|-------------|
| `@workspace/api-spec` | OpenAPI 3.1 spec (`openapi.yaml`) and Orval codegen configuration |
| `@workspace/api-client-react` | Generated React Query hooks and fetch client |
| `@workspace/api-zod` | Generated Zod validation schemas from the OpenAPI spec |
| `@workspace/db` | Drizzle ORM schema definitions and PostgreSQL client |

## Key design principles

**Single source of truth for the API** — The OpenAPI spec in `lib/api-spec/openapi.yaml` is the authoritative definition. All Zod schemas and React Query hooks are auto-generated from it via Orval. Never edit generated files directly.

**Type safety at every layer** — From database schema (Drizzle) to API validation (Zod) to frontend data fetching (React Query + TypeScript), the entire stack is strongly typed.

**Monorepo with composite TypeScript** — Each package uses TypeScript's `composite` mode and project references so cross-package imports resolve correctly. Always typecheck from the root.
