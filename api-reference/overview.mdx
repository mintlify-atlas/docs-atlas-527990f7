---

The EduLearn Hub API is a REST API that returns JSON responses. It is built spec-first: the OpenAPI specification in `lib/api-spec/openapi.yaml` is the single source of truth, and all server validation, client hooks, and Zod schemas are generated from it.

## Key details

| Property | Value |
|---|---|
| Version | `0.1.0` |
| Base URL | `/api` |
| Response format | `application/json` |
| Auth required | No (v0.1.0) |

## Spec-first approach

All code artifacts are generated from the OpenAPI spec using [Orval](https://orval.dev):

- **`@workspace/api-zod`** — Zod validation schemas used by the API server at runtime
- **`@workspace/api-client-react`** — React Query hooks and TypeScript types for frontend consumers

To view the full specification, open:

```
lib/api-spec/openapi.yaml
```

## Available endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/api/healthz` | Returns server health status |
